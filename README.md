## Next.js App Router Course - Starter

This is the starter project for the [Next.js App Router Course](https://nextjs-app-router-course.vercel.app/).

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

## Features

- [x] Routing with Next.js
- [x] Dynamic Routes
- [x] Nested Routes
- [x] Link Component
- [x] Image Component
- [x] Head Component
- [x] Custom Document
